-----------------
Steps to Run
-----------------
1- Install QtDesing (http://qt-project.org/downloads)
(the icon of the Qtdesigner is at C:\Qt\5.3\msvc2013_64\bin )
2- Install Pyside (pip install PySide)
3- If you want to convert you .ui design to a
python script go to the terminal a type
pyside-uic -o output.py inteface.ui 
pyside-uic can be found at you interpreter folder
at Scripts. Just add the path to the enviroment
variables.
*i am using Anaconda as interpreter
